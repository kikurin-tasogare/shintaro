# 真太郎 Bulk Log 🌿

大切な心友の増量を、食事・筋トレ・体重トレンドの3方向から支えるローカルアプリです。
素材中心・グルテンフリー優先の献立を、実際の食材重量から計算して提案します。

## 機能

### 📝 記録
- **筋トレ記録** — 種目・重量・セット・レップ・RIRを記録。次回の目安を表示
- **食事記録** — カロリー・PFC（タンパク質/炭水化物/脂質）・素材中心フラグ
- **体重記録** — 推移グラフ付き。増量ペースの確認に

### 📋 メニュー作成
- **週間筋トレメニュー** — 週2〜5日の生活リズムに合わせて自動設定
- **毎日の食事提案** — GF優先の16メニューからPFCに近い組み合わせを日替わり提案
- **買い物リスト** — 食事メニューから自動生成

### 🎯 増量特化
- TDEE + 設定したカロリーサープラスの自動計算
- タンパク質目標（体重 × 1.6〜2.2g）の自動設定
- 目標体重までの進捗表示
- 7日平均体重と前週比を表示

### 🌾 GF・素材中心
- 小麦を使わない主食・主菜を中心に構成
- 包装食品は「表示確認」を明示
- 栄養値は「日本食品標準成分表（八訂）増補2023年」を基準に計算
- グルテンフリーの厳格度を3段階で設定

## 起動方法

```bash
# 方法1: 起動スクリプト
chmod +x start.sh
./start.sh

# 方法2: 直接
python3 -m http.server 8080
```

ブラウザで **http://localhost:8080** を開いてください。

スマホから使う場合は、同じWi-Fi内でPCのIPアドレス（例: `http://192.168.1.10:8080`）にアクセス。

## データ保存

すべてのデータはブラウザの **localStorage** に保存されます。  
設定画面からJSON形式でエクスポート/インポート可能です。

## ファイル構成

```
shintaro/
├── index.html      # メインHTML
├── css/app.css     # スタイル
├── js/
│   ├── app.js      # アプリ本体
│   ├── data.js     # 食材・テンプレートデータ
│   └── storage.js  # データ永続化
├── start.sh        # 起動スクリプト
└── README.md
```

## カスタマイズ

- **設定画面** でプロフィール（体重・身長・目標等）を入力
- `js/data.js` に食材やGF食事テンプレートを追加可能
- `js/data.js` の `WORKOUT_TEMPLATES` に筋トレ種目を追加可能
