# 心友 Bulk Log 🌿

増量（体重増加）に特化した、筋トレ・食事の記録＆メニュー管理アプリです。  
オーガニック食材を優先した食事プラン作成に対応しています。

## 機能

### 📝 記録
- **筋トレ記録** — 種目・セット・レップ・時間を記録。テンプレートからワンタップ入力
- **食事記録** — カロリー・PFC（タンパク質/炭水化物/脂質）・オーガニックフラグ
- **体重記録** — 推移グラフ付き。増量ペースの確認に

### 📋 メニュー作成
- **週間筋トレメニュー** — Push/Pull/Legs等のテンプレート。増量向け3分割の自動設定
- **週間食事メニュー** — オーガニック食材ベースの増量食事テンプレート8種
- **買い物リスト** — 食事メニューから自動生成

### 🎯 増量特化
- TDEE + カロリーサープラス（+300〜500kcal）の自動計算
- タンパク質目標（体重 × 1.6〜2.2g）の自動設定
- 目標体重までの進捗表示
- 増量のコツを毎日表示

### 🌿 オーガニック対応
- 30種以上のオーガニック食材データベース
- すべてオーガニック食材の食事テンプレート
- 購入先ガイド（コープデリ、自然派クラブ等）

## 起動方法

```bash
# 方法1: 起動スクリプト
chmod +x start.sh
./start.sh

# 方法2: 直接
python3 -m http.server 8080
```

ブラウザで **http://localhost:8080** を開いてください。

### スマホで使う
- **GitHub Pages**（公開後）: リポジトリの Settings → Pages で有効化すると `https://<username>.github.io/shintaro/` でアクセス可能
- **ホーム画面に追加**: PWA対応済み。Safari/Chromeの「ホーム画面に追加」でアプリのように使えます
- **ローカルネットワーク**: 同じWi-Fi内でPCのIPアドレス（例: `http://192.168.1.10:8080`）にアクセス

## データ保存

すべてのデータはブラウザの **localStorage** に保存されます。  
設定画面からJSON形式でエクスポート/インポート可能です。

## ファイル構成

```
shintaro/
├── index.html      # メインHTML
├── manifest.json   # PWAマニフェスト
├── sw.js           # Service Worker（オフライン対応）
├── icons/          # PWAアイコン
├── css/app.css     # スタイル
├── js/
│   ├── app.js      # アプリ本体
│   ├── data.js     # 食材・テンプレートデータ
│   └── storage.js  # データ永続化
├── start.sh        # 起動スクリプト
└── README.md
```

## カスタマイズ

- **設定画面** でプロフィール（体重・身長・目標等）を入力
- `js/data.js` にオーガニック食材や食事テンプレートを追加可能
- `js/data.js` の `WORKOUT_TEMPLATES` に筋トレ種目を追加可能
